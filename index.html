ChessMate

A small chess portal project built using MERN (MongoDB, Express, React, Node.js) which also utilizes GraphQL for testing CRUD. ChessMate allows users to watch high rank, live blitz games from the homepage. When logged in users, can practice chess against a CPU opponent and gain access to an array of chess resources.

Tech Blog

A tech blog created using Handlebars and Express. Features full CRUD posting, commenting and login/signup system. Users can make blog posts from their account, edit these posts or delete them if they like. Other users can view and comment on these posts if they like. Backend built with Node/Express/mySQL.

DeckedOut Study Helper

A group project that is similar to Quizlet. Users can create accounts, log in and create their own decks of flashcards for study. Handled the handlebars routing as well as CRUD functionality. I handled the majority of the backend SQL/express routing for the team.

Browser Note Taker

A note-taking application built off of Express.js, users can add, view and delete notes with ease. Leverages local storage to retain notes throughout. Simple to use and boasts a sleek, reactive design.
